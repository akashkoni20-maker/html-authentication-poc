# HTML Authentication POC

This project demonstrates a simple authentication flow using only HTML.

## Pages Included
- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features
- Proper redirections using anchor tags
- No CSS or JavaScript
- Pure HTML structure

## Flow
Login → Dashboard  
Register → Login  
Forgot Password → Login  
Reset Password → Login  
Logout → Login
